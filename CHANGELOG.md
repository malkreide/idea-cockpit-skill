# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-03-14

### Added
- Initial release of the Idea Cockpit Skill
- Maturity model with 5 stages: Spark, Concept, Plan, Active, Frozen
- Fast capture workflow (under 30 seconds, 2 questions only)
- Weekly Ritual workflow (~15 minutes)
- Promote / Freeze workflow
- Quarterly Spark Review workflow
- Direct Notion integration via MCP
- Hard 3-Active limit enforcement
- Bilingual documentation (English / German)
- Anti-elaboration principle: Capture first, develop later
